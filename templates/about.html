About This Project

This Credit Card Fraud Detection System is built using Machine Learning to identify fraudulent transactions in real-time. The system analyzes transaction patterns and predicts whether a transaction is safe or risky.

🚀 Features

  • Real-time fraud prediction
  • Machine Learning based model
  • User-friendly interface
  • Fast and accurate results

🛠️ Technologies Used

  • Python as programming language
  • Machine Learning model XGBClassifier for detect fraud
  • HTML,CSS for frontend design
  • flask using as Backend
  • MySQL using in Database

🎯 Purpose

The main goal of this project is to prevent financial fraud and enhance security in digital transactions using Machine learning model & intelligent systems.
